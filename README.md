Interactive product showcase for the Watchdog platform with guided paths, deployment snippets, and docs-first navigation.

## Local Development

```bash
cd showcase
npm install
npm run dev
```

## What’s Included

- Path-based walkthrough (understand, use, docs)
- Legal gate and slide navigation UX
- Theme toggle with persistent dark/light mode
- Installation command snippets and copy shortcuts
